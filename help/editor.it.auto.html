Tasti di scelta rapida dell'editor
Trova / Sostituisci
keypress Comando
Ctrl + F Trova
Ctrl + H Sostituire
Ctrl + ⇧ + R Sostituisci tutto
Ctrl + G Trova il prossimo
Ctrl + ⇧ + G Trova precedente
Alt + G Vai alla linea
Manipolazione del testo
keypress Comando
Ctrl + Alt + W Attiva / disattiva avvolgimento morbido
Ctrl + KU Trasforma in maiuscolo
Ctrl + KL Trasforma in minuscolo
La modifica
keypress Comando
Esc, Esc Chiudi la finestra dell'editor attivo
Ctrl + Esc, Esc Riduci a icona la finestra dell'editor attivo
Alt + ▵ Massimizza la finestra dell'editor attivo
Alt + ▿ Normalizza la finestra dell'editor attivo
Ctrl + S Salva
Ctrl + ⇧ + S Salva come..
Ctrl + Alt + ⇧ + S Salva e chiudi
Alt + S Apri la ricerca nei file
Ctrl + Space Attiva il completamento automatico
Ctrl + X Linea di taglio
Ctrl + ⇧ + ▵ Sposta linea / selezione in alto
Ctrl + ⇧ + ▿ Sposta linea / selezione in basso
Ctrl + L Seleziona riga: ripeti per selezionare le righe successive
Ctrl + D Seleziona parola: ripeti per selezionare altre occorrenze
Ctrl + M Passa alle parentesi chiuse - Ripeti per passare alle parentesi aperte
Ctrl + ⇧ + M Seleziona tutti i contenuti delle parentesi correnti
Ctrl + ⇧ + K Elimina riga
Ctrl + KK Elimina dal cursore alla fine della riga
Ctrl + K +
Backspace		 Elimina dal cursore all'inizio della riga
Ctrl + ⇧ + D Linee duplicate
Ctrl + J Unisci la riga sotto alla fine della riga corrente
Ctrl + / Riga corrente di commento / non commento
Ctrl + ⇧ + / Blocca la selezione corrente del commento
Ctrl + Y Ripeti o ripeti l'ultimo comando di scelta rapida da tastiera
Ctrl + ⇧ + V Incolla e rientra correttamente
Ctrl + Space Seleziona il suggerimento di completamento automatico successivo
Ctrl + U Annulla morbido; passa all'ultima modifica prima di annullare la modifica quando viene ripetuta
