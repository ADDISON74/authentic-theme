Нажатие клавиши команда
Ctrl + F найти
Ctrl + H замещать
Ctrl + ⇧ + R Заменить все
Ctrl + G Найти следующий
Ctrl + ⇧ + G Найти предыдущий
Alt + G Перейти к строке

Нажатие клавиши команда
Ctrl + Alt + W Переключить мягкую обертку
Ctrl + KU Преобразовать в верхний регистр
Ctrl + KL Преобразовать в нижний регистр
Редактирование
Нажатие клавиши команда
Esc, Esc Закрыть окно активного редактора
Ctrl + Esc, Esc Свернуть активное окно редактора
Alt + ▵ Развернуть окно активного редактора
Alt + ▿ Окно нормализовать активный редактор
Ctrl + S Сохранить
Ctrl + ⇧ + S Сохранить как..
Ctrl + Alt + ⇧ + S Сохранить и закрыть
Alt + S Открыть поиск по файлам
Ctrl + Space Активировать автозаполнение
Ctrl + X Линия разреза
Ctrl + ⇧ + ▵ Переместить строку / выделение вверх
Ctrl + ⇧ + ▿ Переместить строку / выделение вниз
Ctrl + L Выбрать строку - повторите, чтобы выбрать следующие строки
Ctrl + D Выберите слово - Повторите, выберите другие вхождения
Ctrl + M Перейти к закрывающим скобкам - повторите, чтобы перейти к открывающим скобкам
Ctrl + ⇧ + M Выбрать все содержимое текущих скобок
Ctrl + ⇧ + K Удалить строку
Ctrl + KK Удалить от курсора до конца строки
Ctrl + K +
Backspace		 Удалить от курсора до начала строки
Ctrl + ⇧ + D Дублирующая строка (и)
Ctrl + J Присоединить строку ниже к концу текущей строки
Ctrl + / Комментировать / не комментировать текущую строку
Ctrl + ⇧ + / Блок комментариев текущий выбор
Ctrl + Y Повторите или повторите последнюю команду сочетания клавиш
Ctrl + ⇧ + V Вставьте и сделайте отступ правильно
Ctrl + Space Выберите следующее автозаполнение предложения
Ctrl + U Мягкая отмена; переход к последнему изменению перед отменой изменения при повторении
