Atalhos do teclado do editor
Localizar / substituir
Pressione o botão Comando
Ctrl + F Encontrar
Ctrl + H Substituir
Ctrl + ⇧ + R Substitua tudo
Ctrl + G Encontre o próximo
Ctrl + ⇧ + G Localizar anterior
Alt + G Ir para a linha
Manipulação de texto
Pressione o botão Comando
Ctrl + Alt + W Alternar quebra automática
Ctrl + KU Transformar em maiúsculas
Ctrl + KL Transformar em minúsculas
Edição
Pressione o botão Comando
Esc, Esc Feche a janela ativa do editor
Ctrl + Esc, Esc Minimizar a janela ativa do editor
Alt + ▵ Maximizar janela ativa do editor
Alt + ▿ Janela Normalizar editor ativo
Ctrl + S Salve 
Ctrl + ⇧ + S Salvar como..
Ctrl + Alt + ⇧ + S Salvar e fechar
Alt + S Abrir pesquisa em arquivos
Ctrl + Space Ativar o preenchimento automático
Ctrl + X Linha de corte
Ctrl + ⇧ + ▵ Mover linha / seleção para cima
Ctrl + ⇧ + ▿ Mover linha / seleção para baixo
Ctrl + L Selecionar linha - repita para selecionar as próximas linhas
Ctrl + D Selecionar palavra - Repita a seleção de outras ocorrências
Ctrl + M Ir para parênteses de fechamento - Repita para pular para parênteses de abertura
Ctrl + ⇧ + M Selecionar todo o conteúdo dos parênteses atuais
Ctrl + ⇧ + K Excluir linha
Ctrl + KK Excluir do cursor até o final da linha
Ctrl + K +
Backspace		 Excluir do cursor para o início da linha
Ctrl + ⇧ + D Linha (s) duplicada (s)
Ctrl + J Associe a linha abaixo ao final da linha atual
Ctrl + / Comentar / cancelar o comentário da linha atual
Ctrl + ⇧ + / Bloquear comentário seleção atual
Ctrl + Y Refazer ou repetir o último comando de atalho do teclado
Ctrl + ⇧ + V Colar e recuar corretamente
Ctrl + Space Selecione a próxima sugestão de preenchimento automático
Ctrl + U Desfazer suave; pula para sua última alteração antes de desfazer a alteração quando repetida
