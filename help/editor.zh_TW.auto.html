編輯器鍵盤快捷鍵
查找/替換
按鍵命令
Ctrl + F找
Ctrl + H更換
Ctrl + ⇧ + R全部替換
Ctrl + G找下一個
Ctrl + ⇧ + G查找上一個
Alt + G跳線
文字處理
按鍵命令
Ctrl + Alt + W切換軟包裝
Ctrl + KU轉換為大寫
Ctrl + KL轉換為小寫
編輯中
按鍵命令
Esc, Esc關閉活動的編輯器窗口
Ctrl + Esc, Esc最小化活動編輯器窗口
Alt + ▵最大化活動編輯器窗口
Alt + ▿標準化活動編輯器窗口
Ctrl + S救
Ctrl + ⇧ + S另存為..
Ctrl + Alt + ⇧ + S保存並關閉
Alt + S在文件中打開搜索
Ctrl + Space激活自動補全
Ctrl + X切割線
Ctrl + ⇧ + ▵向上移動行/選擇
Ctrl + ⇧ + ▿向下移動行/選擇
Ctrl + L選擇行-重複以選擇下一行
Ctrl + D選擇單詞-重複選擇其他出現的單詞
Ctrl + M跳轉到右括號-重複以跳轉到右括號
Ctrl + ⇧ + M選擇當前括號的所有內容
Ctrl + ⇧ + K刪除行
Ctrl + KK從光標刪除到行尾
Ctrl + K +
Backspace		從光標刪除到行首
Ctrl + ⇧ + D重複的行
Ctrl + J將下面的行連接到當前行的末尾
Ctrl + /註釋/取消註釋當前行
Ctrl + ⇧ + /阻止評論當前選擇
Ctrl + Y重做，或重複上一個鍵盤快捷鍵命令
Ctrl + ⇧ + V正確粘貼和縮進
Ctrl + Space選擇下一個自動完成建議
Ctrl + U軟撤消；跳到上一次更改，然後重複更改
